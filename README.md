# Chandru Mobile Shop Website

A modern, responsive website for a mobile shop featuring products and services with beautiful design and sharing capabilities.

## Features

- **Product Showcase**: Display headsets, mobile cases, and screen protectors
- **Service Section**: Mobile repair services including display repair, board work, battery replacement, and more
- **Share Functionality**: Easy sharing via WhatsApp, Facebook, or copy link
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradients, animations, and smooth transitions
- **Interactive Elements**: Hover effects, smooth scrolling, and animated cards

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript for interactivity and functionality

## How to Use

1. Open `index.html` in a web browser
2. The website is fully functional and ready to use
3. Customize the contact information, prices, and product details as needed

## Customization

### Update Contact Information
Edit the contact section in `index.html`:
- Address
- Phone number
- Email
- Working hours

### Update Product Images
Replace the Unsplash image URLs in `index.html` with your own product images:
```html
<img src="YOUR_IMAGE_URL" alt="Product Name">
```

### Modify Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

## Sharing

The website includes a share button that allows customers to:
- Share via WhatsApp
- Share via Facebook
- Copy link to clipboard
- Use native share API on mobile devices
